# Sistemas Operacionais 🖥️

Estudos teóricos e práticos sobre o funcionamento do Kernel, gerenciamento de recursos e sistemas de arquivos.

## 🔍 Foco do Estudo:
- Gerenciamento de Processos e Threads.
- Gerenciamento de Memória (Memória Virtual, Paginação).
- Sistemas de Arquivos e E/S (Entrada e Saída).
- Comandos de terminal (Linux/Bash).

## ⚙️ Ambiente de Testes
- Máquinas Virtuais (VirtualBox/VMware)
- Terminal Linux
